# Sécurité et signalement d’un vélo endommagé

## Vérification avant le départ

Avant de démarrer un trajet, vérifiez rapidement :

- le fonctionnement des freins ;
- l’état visible des pneus ;
- la fixation de la selle ;
- l’état du guidon ;
- l’absence de pièce détachée ;
- l’absence de dommage manifeste sur le cadre.

Si un élément essentiel semble défectueux, n’utilisez pas le vélo.

## Dommages nécessitant une immobilisation

Le vélo ne doit pas être utilisé lorsque :

- un frein ne fonctionne pas ;
- un pneu est fortement dégonflé ou endommagé ;
- le guidon est instable ;
- une roue est déformée ;
- le cadre présente une fissure visible ;
- le verrou empêche une utilisation sûre ;
- un câble électrique est exposé sur un modèle électrique.

## Comment signaler un problème ?

Dans l’application :

1. ouvrez la fiche du vélo ;
2. choisissez **Signaler un problème** ;
3. sélectionnez la catégorie ;
4. décrivez brièvement le dommage ;
5. ajoutez une photographie si cela peut aider ;
6. validez le signalement.

Un signalement ne garantit pas que le vélo sera immédiatement retiré de la flotte. Le statut opérationnel doit être confirmé par le système de flotte.

## Incident pendant un trajet

En cas de problème mécanique pendant un trajet :

- arrêtez-vous dans un endroit sûr ;
- ne poursuivez pas le trajet si la sécurité est compromise ;
- signalez le problème ;
- contactez le service à la clientèle si le trajet ne peut pas être terminé normalement.

En cas d’accident ou de danger immédiat, contactez d’abord les services d’urgence compétents.

## Informations à ne pas inventer

Le service à la clientèle ne doit pas affirmer qu’un vélo a été immobilisé, réparé ou retiré de la flotte sans confirmation du système opérationnel.
