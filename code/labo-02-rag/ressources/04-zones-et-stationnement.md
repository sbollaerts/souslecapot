# Zones de service et règles de stationnement

## Zone de service

Les vélos Bikaroo peuvent circuler librement sur les voies où leur usage est autorisé.

Un trajet doit toutefois être terminé dans une zone de stationnement acceptée par l’application.

La carte de l’application constitue la référence opérationnelle pour les zones actives.

## Types de zones

L’application peut afficher :

- une zone autorisée ;
- une zone recommandée ;
- une zone temporairement indisponible ;
- une zone interdite ;
- une zone soumise à des restrictions horaires.

Les zones peuvent évoluer. Ce document explique les règles générales, mais ne remplace pas la carte opérationnelle à jour.

## Stationnement correct

Le vélo doit être laissé :

- sans bloquer un trottoir ;
- sans gêner une entrée ou une sortie ;
- hors des bandes podotactiles ;
- sans bloquer un passage pour personnes à mobilité réduite ;
- sans empiéter sur une voie de circulation ;
- dans la position demandée par l’application.

## Stationnement interdit

Il est interdit de terminer un trajet :

- devant une sortie de secours ;
- sur un passage pour piétons ;
- dans un espace privé non autorisé ;
- à l’intérieur d’un bâtiment ;
- dans une zone signalée comme interdite ;
- à un endroit qui présente un danger évident.

## Photographie de fin de trajet

L’application peut demander une photographie après le stationnement.

La photographie doit montrer :

- le vélo entier ;
- sa position ;
- l’environnement immédiat ;
- l’absence d’obstacle manifeste.

## Frais possibles

Un stationnement incorrect peut entraîner :

- un avertissement ;
- des frais administratifs ;
- des frais de déplacement du vélo ;
- une suspension temporaire du compte en cas de récidive.

Le montant exact dépend de la situation et de la politique en vigueur. Toute contestation suit la procédure décrite dans `06-contestations-et-remboursements.md`.
