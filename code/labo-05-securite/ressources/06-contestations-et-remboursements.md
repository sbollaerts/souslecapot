# Contestations de frais et demandes de remboursement

## Objet

Ce document décrit la manière de déposer et de traiter une contestation relative à des frais Bikaroo.

Il ne garantit pas qu’une demande sera acceptée.

## Motifs recevables

Une demande de révision peut notamment concerner :

- un trajet resté ouvert après restitution ;
- un montant calculé sur une durée manifestement incorrecte ;
- des frais de stationnement contestés ;
- un double débit apparent ;
- un trajet que le membre affirme ne pas avoir effectué ;
- une interruption liée à une défaillance technique documentée.

## Informations nécessaires

La demande doit contenir autant que possible :

- l’identité du membre ;
- la référence du trajet ou de la transaction ;
- la date et l’heure ;
- le montant contesté ;
- une description claire ;
- les pièces utiles ;
- la confirmation que les informations fournies sont exactes.

Une demande incomplète peut nécessiter des informations supplémentaires avant analyse.

## Étapes de traitement

```text
réception de la demande
→ vérification de l’identité
→ collecte des informations
→ consultation des systèmes de référence
→ application des règles
→ décision ou escalade
→ notification du membre
```

Le LLM peut aider à résumer ou préparer le dossier. Il ne prend pas seul la décision financière.

## Remboursement

Un remboursement peut être accordé lorsque l’analyse montre qu’un montant a été facturé à tort.

Selon le moyen de paiement, le délai d’apparition du remboursement peut varier.

Le service à la clientèle ne doit pas promettre une date bancaire précise.

## Cas nécessitant une validation supplémentaire

Une validation supplémentaire est requise lorsque :

- le montant dépasse 75 € ;
- plusieurs contestations similaires concernent le même compte ;
- une fraude est suspectée ;
- les données opérationnelles sont contradictoires ;
- la demande concerne un dommage corporel ;
- la décision nécessite une exception aux règles habituelles.

## Absence de réponse documentaire

Ce document ne contient pas le statut actuel d’une contestation particulière.

Pour connaître l’état d’une demande existante, il faut consulter le système de support avec un identifiant autorisé.
