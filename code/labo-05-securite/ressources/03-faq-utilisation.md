# Questions fréquentes sur l’utilisation de Bikaroo

## Comment démarrer un trajet ?

Ouvrez l’application Bikaroo, sélectionnez un vélo disponible et scannez le code affiché sur le cadre.

Attendez la confirmation dans l’application avant de déplacer le vélo.

## Puis-je réserver un vélo ?

La réservation dépend de la formule choisie.

- Amateur League : pas de réservation anticipée ;
- Semi-Pro League : réservation jusqu’à 15 minutes ;
- Pro League : réservation jusqu’à 30 minutes.

Une réservation expire automatiquement si le trajet n’est pas démarré dans le délai prévu.

## Comment terminer un trajet ?

Placez le vélo dans une zone autorisée, immobilisez-le correctement et engagez complètement le verrou.

Le trajet est terminé uniquement lorsque l’application affiche une confirmation.

Le simple fait de quitter le vélo ne suffit pas à clôturer le trajet.

## Que faire si l’application ne confirme pas la fin du trajet ?

Vérifiez d’abord le verrou, la connexion réseau et la zone de stationnement.

Relancez une seule fois la fin du trajet.

Si le problème persiste, contactez le service à la clientèle et fournissez l’heure, l’emplacement et, si possible, une photographie.

La procédure détaillée se trouve dans `02-procedure-trajet-reste-ouvert.md`.

## Puis-je utiliser le compte d’une autre personne ?

Non. Un compte Bikaroo est personnel.

Le titulaire du compte est responsable des trajets démarrés avec ses moyens d’accès, sauf fraude démontrée ou incident reconnu par Bikaroo.

## Puis-je connaître le nombre de vélos disponibles dans une zone ?

L’application affiche les disponibilités connues au moment de la consultation.

Un document FAQ ne contient pas le nombre actuel de vélos disponibles. Cette information est dynamique et doit être obtenue depuis le système de flotte.

## Puis-je transporter un passager ?

Non. Les vélos Bikaroo sont conçus pour une seule personne, sauf modèle explicitement identifié comme vélo cargo avec siège homologué.

## Que faire si le vélo présente un dommage avant le départ ?

Ne démarrez pas le trajet si le dommage compromet la sécurité.

Signalez le vélo dans l’application en indiquant le type de problème observé.

Consultez également `05-securite-et-signalement.md`.
