# Sous le capot de l'IA

Dépôt officiel accompagnant le livre **Sous le capot de l'IA**.

Ce dépôt rassemble l'ensemble du code source, des ressources et des laboratoires utilisés tout au long du livre. Il a pour objectif de permettre au lecteur de reproduire les exemples, d'expérimenter les concepts présentés et de construire progressivement une application d'IA complète.

> Le livre explique les concepts. Ce dépôt permet de les mettre en pratique.

---

## Objectifs

Au fil des laboratoires, vous construirez progressivement un assistant IA capable de :

- dialoguer avec un modèle de langage local ;
- exploiter un corpus documentaire grâce au RAG (Retrieval-Augmented Generation) ;
- interagir avec des systèmes externes via des tools et MCP (Model Context Protocol) ;
- orchestrer plusieurs étapes d'un processus métier ;
- résister à quelques attaques classiques (prompt injection, données non fiables, etc.) ;
- produire des traces permettant de comprendre et diagnostiquer son comportement.

L'ensemble du parcours s'appuie sur **Bikaroo**, une entreprise fictive de vélos partagés servant de fil rouge au livre.

---

## Contenu du dépôt

Le dépôt contiendra notamment :

- les laboratoires Python ;
- les laboratoires .NET ;
- les ressources communes (documents, données, images, prompts, etc.) ;
- les versions **startup** et **final** de chaque laboratoire ;
- les ressources nécessaires à la reproduction des exemples du livre.

Chaque laboratoire reprend l'état final du laboratoire précédent afin de construire progressivement une solution plus riche.

---

## Prérequis

Les instructions d'installation de l'environnement de développement sont décrites dans le livre, dans la side note :

> **Mise en place de l'environnement de développement**

Les laboratoires utilisent exclusivement des outils disponibles publiquement et peuvent être exécutés localement.
